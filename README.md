# 🦆 DuckDB

Exploración y uso de DuckDB desarrollado por **Isaac Esteban Haro Torres**.

---

## 📝 Descripción

Este repositorio contiene ejemplos y scripts para explorar las capacidades de DuckDB, una base de datos analítica en proceso (OLAP) incrustada y de alto rendimiento.

## ✨ Características

- Análisis rápido sobre archivos CSV, Parquet
- Integración con Python
- Base de datos embebida ligera

- DuckDB
- Jupyter Notebook
